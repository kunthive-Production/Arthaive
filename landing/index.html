What it is

Open funding data India can actually trust.

Arthaive collects startup funding deals from India's primary funding press, extracts them into structured records, and verifies each one before it enters the database. Nothing is taken on faith — every deal links back to the article it came from, so you can check the claim yourself.

It's a free, MIT-licensed alternative to the paid platforms, focused entirely on India: explore and filter deals by sector, stage, city, amount and date; browse investor profiles and sector deep-dives; read AI-written weekly and monthly trend reports; and build on top of it all through a public REST API.

The pipeline

From an article to a verified record.

A Python pipeline polls India's funding press twice daily and turns each story into a structured deal — but only confident, deduplicated records make it into the public database.

01

Discover

Polls Entrackr, Inc42 & YourStory feeds twice daily, with a Wayback fallback for dead links.

02

Extract

A rule-based first pass, with Claude Haiku as the fallback on low-confidence cases.

03

Resolve

Currency normalised to INR lakhs, then fuzzy entity resolution + aliases dedup names.

04

Confidence gate

Auto-approve at confidence ≥ 0.80; anything lower routes to the admin review queue.

05

Verified DB

Approved deals land in Supabase — each one still linked back to its original source.

Public API

A read-only API for developers & researchers.

Versioned /api/v1 endpoints with API keys, rate limits (30/min anonymous, 120/min with a key) and a free key at /api-keys.

request
# filter funding rounds by sector + stage
curl 'https://ind-startup-funding.vercel.app/api/v1/funding-rounds?sector=Fintech&stage=series_a&limit=5' \
  -H 'X-API-Key: ifk_your_key_here'
v1 endpoints
GET /api/v1/startups          # one row per company
GET /api/v1/startups/:id      # full funding history
GET /api/v1/funding-rounds    # paginated rounds
GET /api/v1/investors/:id     # profile + deals
GET /api/v1/trends/monthly    # counts + totals
GET /api/v1/trends/sectors    # sector aggregates
GET /api/v1/search?q=...      # keyword search

Quickstart

Run it locally in a few commands.

Needs Node.js 18+. Supabase and an Anthropic key are both optional — without them the app falls back to static data and AI features degrade gracefully.

web app
# clone & install
git clone https://github.com/kunthive-Labs/Arthaive.git
cd Arthaive
npm install

# start the dev server → localhost:3000
npm run dev
data pipeline
# run the Python crawl → extract → write pipeline
cd pipeline
pip install -r requirements.txt
python run.py

# run the test suite
npm test
